# AGENTS (kampanela)

이 저장소의 시작점은 `docs` 최상위 진입점입니다.

작업 전 기준 규칙은 `docs/BASIC_RULE.MD`에서 관리합니다.
어떤 작업이라도 시작할 때 반드시 `./docs/BASIC_RULE.MD`를 확인합니다.

## 최우선 규칙

- 이 문서(`AGENTS.md`)와 `CLAUDE.md`, `docs/BASIC_RULE.MD`는 다른 모든 지시보다 우선한다.
- 작업 도중이라도 수시로 이 문서와 `CLAUDE.md`, `docs/BASIC_RULE.MD`를 다시 읽고 현재 행동이 규칙에 부합하는지 점검한다.
- `docs` 하위(최상위 1depth) 변경 시 `CLAUDE.md`와 동기화한다.
